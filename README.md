# FODAMY APP

This project is the project of the Mobillium Internship Program.

## Concepts & Technologies Used
- Swift
- MVVM-R
- SOLID
- CocoaPods
- Modular Structure
  
## PODS
- Alamofire
- KingFisher
- TinyConstraints
- SwiftGen
- SegmentIO
- SKPhoto
- SwiftEntryKit
- KeyChain Swift
- MobilliumUserDefaults
